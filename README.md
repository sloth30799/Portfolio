# Portfolio

An editorial portfolio for Han Ye Htun, a software engineer building expressive interfaces and practical product systems.

## Development

Requires Node.js `^20.19.0 || >=22.12.0` and pnpm `11.9.0`.

```sh
pnpm install --frozen-lockfile
pnpm test:unit
pnpm build
pnpm preview --host 127.0.0.1 --port 4173
```

Open the loopback preview at `http://127.0.0.1:4173`.
